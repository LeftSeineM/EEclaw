# EEclaw 项目说明
项目简介：EEclaw 是清华大学AI学堂桌面客户端，用于管理课程、作业、通知和 AI 智能体。
功能特性：EE学堂、EE智能体、EE选课、设置等。

## 如何在本地运行
1. 克隆仓库：`git clone https://github.com/LeftSeineM/EEclaw.git`
2. 安装依赖：`npm install`
3. 启动项目：`npm run dev`

## 如何下载安装包（better）
请前往 [Releases](你的仓库链接/releases) 页面下载最新的 .exe 文件。

## exe应用使用说明：
### 登录：
清华学号 + 统一身份认证
### 设置：
主题、**数据存储路径**（推荐前期设置好）
### 配置EE智能体：
推荐在设置中**配置智谱API**，网址：https://open.bigmodel.cn/
（或配置本地Ollama）
